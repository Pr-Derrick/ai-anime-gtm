**4. What failure mode would make this embarrassing or expensive?**

- Character face changes between episodes → audience stops trusting the show
- ElevenLab voiceover sounds robotic → completion rate collapses
- Episode 1 has no hook in the first 3 seconds → platform won’t distribute
- Content flagged for copyright → taken down mid-run

**5. What tiny sample would prove the workflow is worth scaling?**
1 theme → complete 10-step pipeline → 1 published episode → 72-hour data review.
If completion rate > 30% and no production errors, scale to 10 episodes.
